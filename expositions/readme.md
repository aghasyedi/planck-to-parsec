## 📝 Maintenance & Publication Guide

To maintain the structural integrity and academic rigor of this directory, please adhere to the following conventions when archiving new manuscripts:

1. **Serial Numbering:** Always use a 4-digit, zero-padded serial number (`0000`, `0001`, `0002`, `0010`, etc.) in the `Index` column to ensure chronological scalability.
2. **File Naming:** Ensure the file names in the `File Link` column exactly match the uploaded files (including capitalization). Use `%20` for spaces in URLs (e.g., `Dark%20Matter%20Candidates.pdf`).
3. **Descriptions:** Keep the descriptions concise to ensure the table renders efficiently on smaller screens.
4. **Rigorous Verification:** In accordance with the repository's methodology, ensure all synthesized frameworks and theoretical extrapolations in new uploads have undergone strict manual verification before being indexed here.
