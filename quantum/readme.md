## 📝 Maintenance Guide

To keep this directory organized, please adhere to the following conventions when adding new files:

1. **Serial Numbering:** Always use a 4-digit, zero-padded serial number (`0000`, `0001`, `0002`, `0010`, etc.) in the `Index` column.
2. **File Naming:** Ensure the file names in the `File Link` column exactly match the uploaded files (including capitalization). Use `%20` for spaces in links (e.g., `Squeezed%20Light.pdf`).
3. **Descriptions:** Keep the descriptions concise to ensure the table renders well on smaller screens and mobile devices.
